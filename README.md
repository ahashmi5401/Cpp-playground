## 🚀 Getting Started

### Prerequisites — Install a C++ Compiler

```bash
# Check if already installed
g++ --version

# Ubuntu / Debian
sudo apt install g++ build-essential

# macOS
brew install gcc

# Windows → Install MinGW: https://www.mingw-w64.org/
```

---

## 🔨 Compile & Run

### Basic

```bash
g++ filename.cpp -o output
./output          # Linux / macOS
output.exe        # Windows
```

### Recommended (C++17 + Warnings)

```bash
g++ -std=c++17 -Wall -Wextra filename.cpp -o output
./output
```

### Clone & Run Any File

```bash
git clone https://github.com/ahashmi5401/Cpp--Playground-
g++ -std=c++17 hello_world.cpp -o hello && ./hello
```

### Compiler Flags

| Flag | What it does |
|------|-------------|
| `-std=c++17` | Modern C++17 standard |
| `-Wall` | Enable all common warnings |
| `-Wextra` | Enable extra warnings |
| `-g` | Debug info for GDB |
| `-O2` | Performance optimization |
